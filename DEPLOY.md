# Как выложить X-Raya в интернет — пошагово (без терминала)

Цель: получить публичную ссылку, где X-Raya работает у всех, AI прикручен, а ключ скрыт.
Всё делается мышкой на трёх сайтах: Anthropic → GitHub → Vercel. Время ~20 минут.

Папка с проектом на твоём Mac: **Finder → Macintosh HD → Пользователи → macbook → sourcing-agent**
(внутри: `index.html`, папка `api`, `README.md`).

---

## ЧАСТЬ 0. Ключ Anthropic (мозг для AI)

1. Открой https://console.anthropic.com и зарегистрируйся (можно через Google).
2. Слева внизу/в меню → **Billing** (Оплата) → добавь карту и пополни баланс (хватит $5;
   модель Haiku очень дешёвая — тысячи запросов на эти деньги).
3. Слева → **API Keys** → кнопка **Create Key** → имя любое (например `x-raya`) → **Create**.
4. Скопируй ключ (длинная строка, начинается на `sk-ant-...`) и **сохрани в заметку** —
   он показывается один раз. Это понадобится в Части 2.

---

## ЧАСТЬ 1. Загрузить код на GitHub

GitHub — это «облачное хранилище кода», откуда Vercel заберёт сайт.

1. Открой https://github.com → **Sign up** → заведи аккаунт (почта, пароль, подтверждение).
2. После входа справа вверху нажми **«+»** → **New repository**.
3. Заполни:
   - **Repository name:** `x-raya`
   - Поставь точку **Public** (публичный).
   - Ничего больше не трогай.
   - Нажми зелёную кнопку **Create repository**.
4. Откроется страница репозитория. Найди ссылку **«uploading an existing file»**
   (в тексте «Get started by … or uploading an existing file»). Нажми её.
   - Если не видишь ссылку — нажми кнопку **Add file → Upload files**.
5. Теперь открой на Mac папку `sourcing-agent` (Finder).
6. Выдели и **перетащи** в окно браузера (в зону «Drag files here»):
   - файл `index.html`
   - **папку** `api` (целиком — внутри неё `ai.js`)
   - файл `README.md` (по желанию)
   > Важно: папку `api` тащи именно как папку — GitHub сохранит её структуру.
7. Внизу страницы нажми зелёную кнопку **Commit changes**.
8. Готово — на странице репозитория должны появиться `index.html` и папка `api`.

---

## ЧАСТЬ 2. Развернуть на Vercel (сервер + публичная ссылка)

Vercel бесплатно отдаёт сайт и запускает `api/ai.js` как защищённый прокси.

1. Открой https://vercel.com → **Sign Up** → выбери **Continue with GitHub** →
   разреши доступ (**Authorize Vercel**).
2. На дашборде нажми **Add New… → Project**.
3. В списке репозиториев найди **x-raya** → нажми **Import**.
   - Если репозитория нет в списке → **Adjust GitHub App Permissions** / **Configure** →
     дай Vercel доступ к репозиторию x-raya → вернись.
4. На экране настройки проекта:
   - **Framework Preset:** оставь **Other** (само определится).
   - Раскрой блок **Environment Variables** (Переменные окружения) и добавь:
     - **Key (имя):** `ANTHROPIC_API_KEY`  →  **Value (значение):** вставь ключ `sk-ant-...` из Части 0 → **Add**.
     - (по желанию) **Key:** `AI_MODEL` → **Value:** `claude-haiku-4-5-20251001`
       (или `claude-sonnet-4-6`, если хочешь ответы покачественнее и подороже).
   - Нажми **Deploy**.
5. Подожди ~1 минуту. Появится экран **Congratulations** и превью.
   Нажми **Continue to Dashboard** или **Visit** — это твоя публичная ссылка
   вида `https://x-raya.vercel.app`. Делись ей с кем угодно.

> Если блок Environment Variables не появился на шаге импорта — ничего страшно:
> сначала нажми Deploy, потом зайди в **проект → Settings → Environment Variables**,
> добавь там `ANTHROPIC_API_KEY`, и нажми **Deployments → … (три точки) → Redeploy**.

---

## ЧАСТЬ 3. Проверить

1. Открой свою ссылку `https://x-raya.vercel.app`.
2. Введи запрос, например `сеньор python разработчик москва` → **Найти**. Ссылки появятся.
3. На карточке (например LinkedIn) нажми **шире / уже / по-другому** — через пару секунд
   строка и ссылка обновятся (это работает AI через сервер).
4. Поставь галочку **AI-интерпретация** и снова **Найти** — синонимы станут «умнее».

Если AI-кнопки выдают ошибку внизу экрана — почти всегда это: не добавлен `ANTHROPIC_API_KEY`
или не пополнен баланс Anthropic. Проверь Часть 0 и Часть 2, шаг 4.

---

## Как обновлять сайт потом

Любая правка кода = новый деплой автоматически.
- Простой способ: на GitHub открой нужный файл → карандаш ✏️ (Edit) → внеси правки →
  **Commit changes**. Vercel сам пересоберёт сайт за минуту.
- Или снова **Add file → Upload files** и залей новую версию `index.html`.

## Частые вопросы
- **Это платно?** Vercel и GitHub — бесплатно для такого проекта. Платишь только Anthropic
  за запросы AI (центы). Без AI всё работает бесплатно и офлайн.
- **Ключ виден кому-то?** Нет. Он лежит только в настройках Vercel (Environment Variables),
  в код и в браузер не попадает.
- **Хочу свой домен** (например xraya.ru): Vercel → проект → Settings → Domains → Add.
