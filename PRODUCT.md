# X-Raya — PRODUCT.md

## Что это
Сорсинг-агент для рекрутеров: по запросу на естественном языке («сеньор python разработчик Москва»)
генерирует готовые boolean/X-Ray поисковые ссылки по ~30 каналам (LinkedIn, hh.ru, GitHub, Telegram,
VK, TenChat, конференции, соревнования, профильные сайты). IT и non-IT роли.

## Кто пользуется
Рекрутеры и сорсеры (RU-рынок). Работают быстро, многозадачно, часто десятки поисков в день.
Терминология сорсинга им родная: X-Ray, boolean, каналы.

## Задача страницы (одна)
Запрос → набор готовых поисковых ссылок, сгруппированных по каналам, за секунды.

## Регистр
Product (инструмент), но с намеренной брендовой оболочкой: «оперативная разведка талантов» —
хедхантер выслеживает цель. Театральность живёт в раме (шапка, консоль, радар, досье цели);
рабочая зона (карточки запросов, кнопки, фильтры) — дисциплинированная и быстрая.

## Технически
Один файл index.html (vanilla HTML/CSS/JS, без сборки) + server.js (Node, zero-dep).
Шрифты self-hosted в /fonts (РФ без VPN). Деплой: Coolify → x-raya.space, редеплой вручную.
ИИ-слой через /api/ai (OpenRouter, ключ на сервере) — сейчас заблокирован по IP, деградирует мягко.

## Инварианты (не ломать)
- ID: #q, #results, #parsed, #filtCnt, #filters, #aiOn — на них завязан JS.
- Логика parse/build/mergeState/фильтры/openGroup/copyGroup/aiVariant — менять только осознанно.
- Никаких внешних CDN: всё self-hosted (РФ-доступность).
- Словарь ролей ведётся в X-Raya-roles.xlsx → tools/compile.js → apply-titles.js.
